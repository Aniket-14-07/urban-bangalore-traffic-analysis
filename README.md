# Urban Traffic Analysis & ML-Based Prediction System

## Overview
This project analyzes Bangalore traffic data using Machine Learning and statistical techniques to study traffic patterns, predictive behavior, and classification performance.

The workflow includes:
- Exploratory Data Analysis (EDA)
- Regression Analysis
- PCA & MDS for dimensionality reduction
- Classification using Logistic Regression, SVM, and MLP
- Ensemble learning using Random Forest, AdaBoost, and Gradient Boosting

---

## Objectives
- Analyze urban traffic patterns across Bangalore
- Identify important traffic-related indicators
- Apply machine learning techniques for regression and classification
- Compare performance of multiple ML algorithms

- Multi-Dimensional Scaling (MDS)
- Linear Discriminant Analysis (LDA)

---

## Dataset Features
The dataset includes:
- Traffic Volume
- Average Speed
- Travel Time Index
- Incident Reports
- Weather Conditions
- Roadwork Activity
- Pedestrian & Cyclist Count

---

## Key Insights
- Traffic volume varies significantly across Bangalore regions
- Incident reports positively influence congestion patterns
- Dimensionality reduction techniques helped simplify feature relationships
- Ensemble models improved classification performance compared to baseline methods
